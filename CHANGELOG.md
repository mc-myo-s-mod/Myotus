# Changelog

## Feature
- Added a shared Myotus creative tab and a new creative tab registration API for add-on mods via `ICreativeTabRegistrar` and `MyotusAPI.creativeTabRegistrar()`.
- Added documentation for contributing creative tab items and stacks from dependent mods.
- Updated the issue report button to open the Myotus GitHub issues page through Minecraft's external link confirmation screen instead of copying the URL to the clipboard.
- Refreshed the side panel toggle button with dedicated show/hide icons and clearer tooltip text.
