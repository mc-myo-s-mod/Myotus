# Myotus

Myotus is a small extension library for Applied Energistics 2 terminals.

On its own, Myotus keeps the visible gameplay surface intentionally light. Most of its value appears when another mod
uses it to add terminal upgrade cards or terminal-specific behavior.

## Quick Notes

- The terminal settings screen is opened from AE2 terminal screens with the Myotus keybind.
- The Terminal Upgrade Panel can be shown or hidden from the terminal toolbar.
- View Cell slots can be shown in the Myotus Terminal Upgrade Panel on Minecraft 1.21.1.
- Minecraft 1.20.1 keeps the View Cell option visible as a compatibility note, but the option itself is disabled.
