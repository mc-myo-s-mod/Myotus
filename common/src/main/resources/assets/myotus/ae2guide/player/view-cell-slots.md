# View Cell Slots

On Minecraft 1.21.1, Myotus can show AE2 View Cell slots inside the Myotus Terminal Upgrade Panel.

When **Show view cell slots** is enabled, the panel uses two columns:

- the left column shows View Cell slots
- the right column shows terminal upgrade slots

When the option is disabled, View Cell slots are hidden and the panel uses only the upgrade-slot column.

## Minecraft 1.20.1

The View Cell option is visible in the Myotus settings screen on Minecraft 1.20.1, but it is disabled and marked as
1.21.1-only.
