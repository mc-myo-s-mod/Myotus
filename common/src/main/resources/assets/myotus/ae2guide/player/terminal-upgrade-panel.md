# Terminal Upgrade Panel

Myotus adds a Terminal Upgrade Panel to AE2 storage terminals.

The panel is opened with the Myotus button on the terminal toolbar. If the button is hidden in the Myotus settings tab,
the panel can still be opened with the Terminal Upgrade Panel keybind.

## Upgrade Slots

The panel contains five terminal upgrade slots.

These slots are used by Myotus-compatible terminal upgrade cards. Myotus itself is mostly a library, so most useful
cards are expected to come from addons.

Empty upgrade slots show a tooltip listing upgrade cards that can still be installed in the current terminal.

## Saved Upgrades

Installed upgrade cards are saved with the terminal they belong to.

Placed terminal parts keep their installed upgrade cards when they are broken and dropped. Item-backed terminals keep
their upgrade storage tied to the terminal item.
