# Terminal Settings

Myotus adds a small settings tab to AE2 terminal screens.

The default keybind for opening the terminal settings screen is `,`. The keybind only matters while an AE2 terminal
screen is focused.

## Options

**Show upgrade panel button** controls whether the terminal toolbar shows the button that opens the Myotus Terminal
Upgrade Panel.

**Show view cell slots** controls whether the Terminal Upgrade Panel uses a second column for AE2 View Cell slots. This
option works on Minecraft 1.21.1. On Minecraft 1.20.1 it is shown as disabled and marked as 1.21.1-only.
