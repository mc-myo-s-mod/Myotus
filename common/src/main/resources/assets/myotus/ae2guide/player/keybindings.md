# Keybindings

Myotus adds terminal-only keybindings. They are active while an AE2 terminal screen is focused.

## Open Terminal Settings

Default key: `,`

This opens the Myotus tabbed terminal settings screen.

## Toggle Terminal Upgrade Panel

Default key: `.`

This opens or closes the Myotus Terminal Upgrade Panel from the terminal screen.
