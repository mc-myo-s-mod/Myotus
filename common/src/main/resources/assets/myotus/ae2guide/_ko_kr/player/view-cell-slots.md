# View Cell 슬롯

Minecraft 1.21.1에서 Myotus는 AE2 View Cell 슬롯을 Myotus 터미널 업그레이드 패널 안에 표시할 수 있습니다.

**View Cell 슬롯 표시**가 켜져 있으면 패널은 두 열을 사용합니다.

- 왼쪽 열은 View Cell 슬롯입니다.
- 오른쪽 열은 터미널 업그레이드 슬롯입니다.

옵션이 꺼져 있으면 View Cell 슬롯은 숨겨지고 패널은 업그레이드 슬롯 한 열만 사용합니다.

## Minecraft 1.20.1

Minecraft 1.20.1에서도 Myotus 설정 화면에 View Cell 옵션이 보이지만, 비활성화되어 있으며 1.21.1 전용으로
표시됩니다.
