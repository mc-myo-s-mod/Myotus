# 터미널 설정

Myotus는 AE2 터미널 화면에 작은 설정 탭을 추가합니다.

터미널 설정 화면의 기본 키는 `,`입니다. 이 키바인드는 AE2 터미널 화면에 포커스가 있을 때만 의미가 있습니다.

## 옵션

**업그레이드 패널 버튼 표시**는 터미널 툴바에 Myotus 터미널 업그레이드 패널을 여는 버튼을 표시할지 정합니다.

**View Cell 슬롯 표시**는 터미널 업그레이드 패널에 AE2 View Cell 슬롯용 두 번째 열을 표시할지 정합니다. 이
옵션은 Minecraft 1.21.1에서 동작합니다. Minecraft 1.20.1에서는 비활성화된 상태로 표시되며 1.21.1 전용으로
안내됩니다.
