# Myotus

Myotus는 Applied Energistics 2 터미널을 확장하기 위한 작은 라이브러리입니다.

단독으로는 큰 콘텐츠를 추가하지 않습니다. 다른 모드가 Myotus를 이용해서 터미널 업그레이드 카드나 터미널 전용
동작을 추가할 때 가치가 커집니다.

## 빠른 메모

- Myotus 키바인드로 AE2 터미널 안에서 터미널 설정 화면을 열 수 있습니다.
- 터미널 툴바의 버튼으로 Myotus 터미널 업그레이드 패널을 열고 닫을 수 있습니다.
- Minecraft 1.21.1에서는 Myotus 터미널 업그레이드 패널에 View Cell 슬롯을 표시할 수 있습니다.
- Minecraft 1.20.1에서는 View Cell 옵션이 호환성 안내용으로만 보이고 비활성화됩니다.
