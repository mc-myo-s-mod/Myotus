# 터미널 업그레이드 패널

Myotus는 AE2 스토리지 터미널에 터미널 업그레이드 패널을 더합니다.

패널은 터미널 툴바의 Myotus 버튼으로 열 수 있습니다. Myotus 설정 탭에서 버튼을 숨겼더라도 터미널 업그레이드
패널 키바인드로 열 수 있습니다.

## 업그레이드 슬롯

패널에는 터미널 업그레이드 슬롯 5개가 있습니다.

이 슬롯은 Myotus와 호환되는 터미널 업그레이드 카드가 사용합니다. Myotus 자체는 주로 라이브러리이므로, 실제로
유용한 카드는 대부분 애드온 모드에서 제공됩니다.

빈 업그레이드 슬롯은 현재 터미널에 아직 설치할 수 있는 업그레이드 카드를 툴팁으로 보여줍니다.

## 저장되는 업그레이드

설치된 업그레이드 카드는 해당 터미널에 맞춰 저장됩니다.

설치된 터미널 파트는 부서져 드롭될 때 장착된 업그레이드 카드를 유지합니다. 아이템 기반 터미널은 터미널 아이템에
연결된 업그레이드 저장소를 사용합니다.
