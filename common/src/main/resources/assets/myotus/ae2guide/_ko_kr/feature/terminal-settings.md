# 터미널 설정

Myotus는 AE2 `터미널 설정`에 간단한 설정을 할 수 있는 탭을 추가합니다.
또한 Myotus와 관련된 모드들의 설정을 여기서 다룰 수 있습니다. 
<FloatingImage  src="../assets/myotus_tab.png" align="right"/>

## 옵션

<Translate key="gui.myotus.checkbox.show_upgrade_panel_button" bold="true" />는 터미널 툴바에 Myotus 터미널 업그레이드 패널을 여는 버튼을 표시할지 정합니다.

<Translate key="gui.myotus.checkbox.show_view_cell_slots.only" bold="true" />는 터미널 업그레이드 패널에 AE2 View Cell 슬롯용 두 번째 열을 표시할지 정합니다. 이
옵션은 Minecraft 1.21.1에서 동작합니다. Minecraft 1.20.1에서는 비활성화된 상태로 표시되며 1.21.1 전용으로 안내됩니다.


## 키 설정
이 페이지에서 설정한 키 설정들은 설정의 키설정과 동기화가 됩니다.  
기본적으로 터미널 키 설정 들은 터미널이 열려 있을때만 작도합니다.
## 터미널 설정 열기

키: <KeyBind id="key.myotus.open.terminal_setting" />

Myotus 탭이 포함된 터미널 설정 화면을 엽니다.

## 터미널 업그레이드 패널 전환

키: <KeyBind id="key.myotus.toggle.subsidepanel" />

터미널 화면에서 Myotus 터미널 업그레이드 패널을 열거나 닫습니다.
