# Terminal Settings

Myotus adds a tab for simple settings to the AE2 `Terminal Settings` screen.
Settings for Myotus-related mods can also be managed here.
<FloatingImage  src="../assets/myotus_tab.png" align="right"/>

## Options

<Translate key="gui.myotus.checkbox.show_upgrade_panel_button" bold="true" /> controls whether the terminal toolbar shows the button that opens the Myotus Terminal Upgrade Panel.

<Translate key="gui.myotus.checkbox.show_view_cell_slots.only" bold="true" /> controls whether the Terminal Upgrade Panel shows a second column for AE2 View Cell slots. This
option works on Minecraft 1.21.1. On Minecraft 1.20.1, it is shown as disabled and marked as 1.21.1-only.


## Key Settings
The key settings configured on this page are synchronized with Minecraft's key mappings.  
By default, terminal key settings only work while a terminal is open.
## Open Terminal Settings

Key: <KeyBind id="key.myotus.open.terminal_setting" />

Opens the terminal settings screen that includes the Myotus tab.

## Toggle Terminal Upgrade Panel

Key: <KeyBind id="key.myotus.toggle.subsidepanel" />

Opens or closes the Myotus Terminal Upgrade Panel from a terminal screen.
